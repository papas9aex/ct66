Отчет по проекту TaskManagerWithTracing

Функциональность:
- Добавление задач с трассировкой (время начала, завершения, продолжительность, результат).
- Удаление задач с логом и обработкой ошибок.
- Просмотр списка задач.
- Вся активность логируется через класс Trace и Stopwatch.

Пример трассировки:
>>> Начало операции: Add — 2025-04-21 10:00:00
<<< Операция Add выполнена успешно. Время выполнения: 12 мс

Используемые технологии:
- .NET Console Application
- System.Diagnostics.Trace
- System.Diagnostics.Stopwatch
